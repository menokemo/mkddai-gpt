# Security

## Secrets

Never commit:

- OpenRouter keys
- GitHub PAT
- n8n encryption key
- PostgreSQL password
- Open WebUI secret key

## OpenHands

OpenHands can execute commands.

Use strict prompts and workspace boundaries.

## GitHub

Use fine-grained token with minimum required permissions.

## n8n

Keep n8n authenticated.

Use HTTPS later if exposed outside LAN.
